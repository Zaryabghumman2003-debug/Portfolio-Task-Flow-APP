# 📋 TaskFlow - Task Management Web Application

## 🚀 Overview
TaskFlow is a fully functional task management web application built with **HTML, CSS, and JavaScript** with **localStorage** for data persistence.

## ✨ Features

### 🔐 Authentication
- Signup with Email, Username, Password
- Login with Username & Password
- Social Login (Google & Facebook - Demo)
- Password visibility toggle
- Remember me functionality
- Terms & Conditions

### 📊 Dashboard
- Statistics Cards (Total, Completed, Pending, Overdue)
- Progress Ring with percentage
- Today's Tasks
- Priority Distribution (High, Medium, Low)
- Workflow Graph (Daily/Weekly/Monthly)
- Live task counters

### ✅ Task Management
- Add Task (Title, Description, Category, Priority, Due Date)
- Edit Task (Inline editing)
- Delete Task (With confirmation)
- Mark as Complete/Incomplete
- Search Tasks (Real-time)
- Filter Tasks (By Status, Priority, Category)

### ⚙️ Additional Features
- Dark/Light Mode Toggle
- Export Data (JSON download)
- Import Data (JSON upload)
- Clear All Tasks (With confirmation)
- Keyboard Shortcuts (Ctrl+N, Ctrl+/, ESC)
- Responsive Design (Mobile, Tablet, Desktop)
- One-click toggle sidebar menu

## 🛠️ Technologies Used
